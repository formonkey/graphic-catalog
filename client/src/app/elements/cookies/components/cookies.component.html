This is a project developed for a technical test using Angular 15 and GraphQL as the connection method with the server. At the top of the browser, you will find a search bar that will allow you to search through the results shown below.

To connect with the server, multiple requests are required without the need to store data in a database. For this purpose, we have created a docker-compose file that can be found in the root of the project. This file can be easily launched using a simple command found in the README file.

                docker-compose up

The docker-compose file includes all the necessary configurations to connect with the server and perform the necessary requests without having to manually configure anything. Additionally, this allows the project to be run in any environment without worrying about dependencies or server configuration.
